# Real-Time Chat Application

A lightweight chat app built with Node.js, Express, and Socket.io that enables real-time messaging without page refreshes.

## Overview

- **Front End:** HTML, CSS, JavaScript  
- **Back End:** Express & Socket.io  
- **Features:** Real-time message updates, auto-scrolling

## Project Structure

WhatsApp/
├── public/
│   ├── index.html      # Main UI file
│   ├── style.css       # Styling rules
│   └── client.js       # Client-side Socket.io & UI logic
├── server.js           # Express server & Socket.io integration
├── package.json        # Project configuration & dependencies
└── README.md           # This file

## Getting Started

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/namaniisc/ChitChat
   
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the App:**
   - For development: `npm run dev` (using Nodemon)
   - Or: `node server.js`
   
4. **Open in Browser:**
   Visit `http://localhost:3000`
